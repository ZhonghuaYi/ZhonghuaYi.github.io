Hexo+Github博客搭建

不久前才通过网上的教程学习搭建了Hexo博客搭建，并且也成功搭建且运行了起来。有朋友提到可以将这个搭建过程写成博客分享出来，于是就有了这一片博文。

下面是关于搭建博客的一些介绍。Hexo是一个基于node.js的一个静态页面框架，可以帮助你快速的搭建博客，并且它配合git+GitHub，可以迅速的将你的本地文件生成HTML并且上传到GitHub；它也提供了优秀的本地server服务，方便用户根据本地渲染出的HTML来修改文章内容或主题配置；另外，它只需要用户提供markdown文件，就可以渲染出样式丰富的HTML。

先介绍搭建hexo博客需要的环境或功能：

  • git/GitHub
  • node.js

以上环境需要你自行配置，可以参考网络上的许多教程。只需要注意，将node.js的根目录路径、git的cmd文件夹路径添加到系统的环境变量中，这样方便在配置过程中使用cmd命令或者git bash。
